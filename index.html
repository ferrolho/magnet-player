What should I write in the input box above?

Find a torrent you want to watch, then copy-paste one of the following into the input box:

  • magnet uri (string)
  • torrent file (buffer)
  • info hash (hex string or buffer)
  • parsed torrent (from parse-torrent)
  • http/https url to a torrent file (string)

Can you give an example?

Sure! Try copy-pasting this https url of a torrent to the input box:

https://webtorrent.io/torrents/sintel.torrent
